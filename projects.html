Research & Academic Projects

The following projects reflect my academic and research experience in digital and analog VLSI design, computer architecture, FPGA-based systems, and hardware acceleration. My work emphasizes rigorous architecture design, timing analysis, and system-level validation.

FPGA-Based Time-to-Digital Converter for Quantum Key Distribution

M.Tech Major Project (Ongoing)
This project focuses on the design of a high-resolution time-to-digital converter (TDC) for quantum key distribution (QKD) receiver systems, where precise photon arrival timing is critical for secure key generation.

  • Implemented SPI-based interfacing and finite state machine control for the TI TDC7200 on FPGA
  • Achieved timing resolution of approximately 55 ps on Basys-3 FPGA platform
  • Currently developing a fully FPGA-based tapped delay-line TDC with coarse–fine time combination
  • Targeting sub-100 ps resolution with long-term goals of ASIC migration

Technologies: Verilog, FPGA, SPI, Quantum Communication

16-bit Multicycle CISC Processor on FPGA

Designed and implemented a 16-bit multicycle CISC processor with a custom instruction set supporting arithmetic operations, memory access, shift/rotate functions, and conditional branching.

  • Implemented unified 32×16-bit memory architecture using distributed RAM
  • Validated design using 8-bit unsigned multiplication test programs
  • Achieved maximum operating frequency of 138 MHz with efficient resource utilization

Technologies: Verilog, Processor Architecture, FPGA

FPGA–CPU Co-Processor for HDR Image Tone-Mapping

Developed a hardware–software co-processing system for HDR image tone-mapping, accelerating computationally intensive Gaussian filtering on FPGA while leveraging CPU-based post-processing.

  • Implemented 2D convolution with a 5×5 Gaussian filter using line-buffer optimization
  • Designed UART-based communication between FPGA and host processor
  • Achieved tone-mapped image quality comparable to software implementations

Technologies: Verilog, FPGA, Image Processing

16-bit Pipelined Radix-4 Booth Multiplier

Designed a two-stage pipelined signed/unsigned radix-4 Booth multiplier optimized for high throughput and energy efficiency in digital arithmetic units.

  • Implemented Wallace tree reduction and carry-skip adders in GPDK45nm technology
  • Optimized MOSFET sizing and arithmetic blocks for performance and power
  • Achieved high operating frequency with reduced area overhead

Technologies: Cadence Virtuoso, Digital VLSI Design
